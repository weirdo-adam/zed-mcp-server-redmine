# Redmine API Coverage

The Redmine tool surface is provided by the standalone `redmine-mcp-server`
executable. This Zed extension only launches that server and maps Zed settings
to environment variables.

Current API coverage is documented in the standalone server repository:

```text
https://github.com/weirdo-adam/redmine-mcp-server/blob/main/docs/api-coverage.md
```
